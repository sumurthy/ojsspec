## resolve<R>

Make a new promise from the thenable. 
A thenable is promise-like in as far as it has a "then" method.

##### Signature
resolve<R>(value?: R | Thenable<R>): Promise<R>

#### Returns
Promise<R>

#### Parameters


| Parameter	   | Type    | Description |
|:-------------|:---------------|:------------|
| value?     | undefined | %optional% undefined |
